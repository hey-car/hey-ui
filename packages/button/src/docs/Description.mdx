A button is used as a trigger to perform a specific action.
The button unambiguously tells the user what will happen after clicking on it.

## Type of the buttons

Several types of buttons (view) are available in the library.

- <b>Primary</b>. Indicates a key action. There can be only one on the screen
  main button.

- <b>Secondary</b>. Secondary button. Can be used for any actions.
