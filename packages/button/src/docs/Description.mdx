import { Container, Row, Col } from 'storybook/blocks';
import { Whatsapp } from '../../../icons/src';
import Button from '../Button';

## Variants

Use the `variant` prop to change the visual style of the Button.
You can set the value to solid, `contained` (default), `outlined`, and `link`.

```tsx live
const buttonVariants = ['contained', 'outlined', 'link'];

render(
  <Container>
    <Row align="middle">
      {buttonVariants.map(variant => (
        <Col key={variant}>
          <Button size="large" variant={variant}>
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
  </Container>,
);
```

## Sizes

Use the size prop to change the size of the button. You can set the value to `large` or `small`.

```tsx live
const buttonSizes = ['large', 'small'];

render(
  <Container>
    <Row style={{ width: '100%' }}>
      {buttonSizes.map(size => (
        <Col span="2" key={size}>
          <Button size={size} variant="contained">
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
    <Row style={{ width: '100%' }}>
      {buttonSizes.map(size => (
        <Col span="2" key={size}>
          <Button size={size} color="primary" variant="outlined">
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
    <Row style={{ width: '100%' }}>
      {buttonSizes.map(size => (
        <Col span="2" key={size}>
          <Button size={size} variant="link">
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
  </Container>,
);
```

## Colors

Use the `color` prop to change the color of the Button.
You can set the value to solid, `primary` (default) and `tertiary`.
For whatsapp button, you can set the color value `whatsapp`

```tsx live
const buttonColors = ['primary', 'tertiary'];

render(
  <Container>
    <Row align="middle">
      {buttonColors.map(color => (
        <Col key={color}>
          <Button size="large" color={color}>
            Button CTA
          </Button>
        </Col>
      ))}
      <Col key="whatsapp">
        <Button size="large" color="whatsapp">
          WhatsApp
        </Button>
      </Col>
    </Row>
    <Row align="middle">
      <Col key="primary">
        <Button size="large" variant="outlined" color="primary">
          Button CTA
        </Button>
      </Col>
    </Row>
  </Container>,
);
```

```tsx live

```

## Button loading state

Pass the `loading` prop to show its loading state. By default, the button will show a spinner and leave the button's width unchanged.

```tsx live
const buttonSizes = ['large', 'small'];

render(
  <Container>
    <Row>
      {buttonSizes.map(size => (
        <Col key={size}>
          <Button loading color="primary" size={size}>
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
    <Row>
      {buttonSizes.map(size => (
        <Col key={size}>
          <Button loading color="primary" variant="outlined" size={size}>
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
  </Container>,
);
```

## Full width Button

Pass the `fullWidth` prop to the button will take up the full width of its container.

```tsx live
const buttonSizes = ['large', 'small'];

render(
  <Container>
    <Row style={{ width: '100%' }}>
      {buttonSizes.map(size => (
        <Col span="4" key={size}>
          <Button fullWidth color="primary" size={size}>
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
    <Row style={{ width: '100%' }}>
      {buttonSizes.map(size => (
        <Col span="4" key={size}>
          <Button fullWidth color="primary" variant="outlined" size={size}>
            Button CTA
          </Button>
        </Col>
      ))}
    </Row>
  </Container>,
);
```
