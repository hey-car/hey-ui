# HeyCar-UIKit - Fonts

The Fonts component is used load Fonts

## Installation

To install and save in your package.json dependencies, run the command below using npm:

```bash
npm install @heycar-uikit/fonts
```

## Usage

```tsx
import fonts from '@heycar-uikit/fonts';
```
