```
import '@heycar-uikit/fonts';

```

or any supported weight

```
import '@heycar-uikit/fonts/200';
import '@heycar-uikit/fonts/300';
import '@heycar-uikit/fonts/500';

```

You can reference the font name in a CSS stylesheet, CSS Module, or CSS-in-JS.

```

body {
  font-family: "Objektiv";
}

p {
     font-family: 'Objektiv';
}

h1 {
     font-family: 'Objektiv';
}
```
