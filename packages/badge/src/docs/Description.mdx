## Variants

Use the `background` prop to change the visual style of the Badge background-color.
You can set the value to `primary`, `secondary`, `tertiary`, `warning`, `highlight`

```tsx live
const badgeVariants = [
  'primary',
  'secondary',
  'tertiary',
  'warning',
  'highlight',
];

render(
  <Container>
    <Row align="middle">
      {badgeVariants.map(variant => (
        <Col key={variant}>
          <Badge text="$250 off" color={variant}>
            ðŸš—
          </Badge>
        </Col>
      ))}
    </Row>
  </Container>,
);
```
