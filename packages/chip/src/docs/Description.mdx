import { Container, Row, Col } from 'storybook/blocks';
import Chip from '../Chip';
import { Dealer, AC } from '../../../icons/src';

## Variants

Use the `variant` prop to use choice or filter chip .
You can set the value to `filter`, `choice`

```tsx live
render(
  <Container>
    <Row align="middle">
      <Col key="filter">
        <Chip variant="filter">Mini</Chip>
      </Col>
      <Col key="choice">
        <Chip variant="choice">Uunder £1,5000</Chip>
      </Col>
    </Row>
  </Container>,
);
```

## Disabled

Use the `disabled` prop to disable the chip it's value is boolean.

```tsx live
render(
  <Container>
    <Row align="middle">
      <Col key="filter">
        <Chip variant="filter" disabled="true">
          Mini
        </Chip>
      </Col>
      <Col key="choice">
        <Chip variant="choice" disabled="true">
          Uunder £1,5000
        </Chip>
      </Col>
    </Row>
  </Container>,
);
```

## Selected

Use the `selected` prop to select the chip choice it's value is boolean.

```tsx live
render(
  <Container>
    <Row align="middle">
      <Col key="filter">
        <Chip variant="filter" selected="true">
          Mini
        </Chip>
      </Col>
      <Col key="choice">
        <Chip variant="choice" selected="true">
          Uunder £1,5000
        </Chip>
      </Col>
    </Row>
  </Container>,
);
```

## Icons

Use the `leftIcon` prop to add icon before choice or filter chip .
You can set the value to any icon

```tsx live
render(
  <Container>
    <Row align="middle">
      <Col key="filter">
        <Chip leftIcon={<Dealer />} variant="filter">
          Mini
        </Chip>
      </Col>
      <Col key="choice">
        <Chip leftIcon={<AC />} variant="choice">
          Uunder £1,5000
        </Chip>
      </Col>
    </Row>
  </Container>,
);
```

## Callbacks

Use the `onDelete` or `onClick` prop to add Callbacks

```tsx live
render(
  <Container>
    <Row align="middle">
      <Col key="filter">
        <Chip
          leftIcon={<Dealer />}
          onDelete={() => alert('onDelete Callback called')}
          variant="filter"
        >
          Mini
        </Chip>
      </Col>
      <Col key="choice">
        <Chip
          leftIcon={<AC />}
          onClick={() => alert('onClick Callback called')}
          variant="choice"
        >
          Uunder £1,5000
        </Chip>
      </Col>
    </Row>
  </Container>,
);
```
