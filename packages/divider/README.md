A divider line separates different content.

## Installation

To install and save in your package.json dependencies, run the command below using npm:

```bash
npm install @heycar-uikit/divider
```

## Usage

```tsx
import Divider from '@heycar-uikit/divider';
function App() {
  render(<Divider />);
}
```
