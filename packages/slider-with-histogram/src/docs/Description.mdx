## Usage

For a basic implementation, pass the number of steps to render the slider with, some initial values and an onChange function.

```tsx live
const leftVal = 0;
const rightVal = 9;
const handleChange = (values, thumbIndex) => {
  console.log(
    `slider values: ${values.join(',')}`,
    `thumb index: ${thumbIndex}`,
  );

  // UPDATE LEFT AND RIGHT VALS HERE
  // ...
};

render(<div></div>);
```
