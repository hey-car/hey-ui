## Max length

The `maxLength` props provide the ability to count and show the length of the content inside the component.

```tsx live
render(
  <Container>
    <Row flex align="top">
      <Col span="8">
        <Textarea
          style={{ width: '300px', height: '90px' }}
          label="Enter Your Message"
          maxLength="200"
          value="Is there any more information that would help the dealer? Let them know as much as possible to help find your perfect wheels quickly"
        />
      </Col>
      <Col span="8">
        <Textarea
          style={{ width: '300px', height: '90px' }}
          label="Enter Your Message"
          maxLength="200"
        />
      </Col>
    </Row>
  </Container>,
);
```

## Disabled

```tsx live
render(

);
```

## Error

The `error` prop toggles the error state and can then be used to provide feedback to the user about the `error`.

```tsx live
render(
  <Container>
    <Row flex align="top">
      <Col span="8">
        <Textarea
          style={{ width: '300px', height: '90px' }}
          error="This is the placeholder for error note"
          label="Enter Your Message"
          value="Is there any more information that would help the dealer? Let them know as much as possible to help find your perfect wheels quickly"
        />
      </Col>
      <Col span="8">
        <Textarea
          style={{ width: '300px', height: '90px' }}
          error="This is the placeholder for error note"
          label="Enter Your Message"
        />
      </Col>
    </Row>
  </Container>,
);
```
