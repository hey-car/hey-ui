Colours are used purposefully to communicate how things function in the interface. This helps us create visual patterns that can make interacting with our product easier and more predictable for the user.

We have carefully selected an accessible color palette that meets the WC3 AAA color contrast standards.

```tsx
// MyApp.js
import '@heycar-uikit/themes/default.css';

function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />;
}

export default MyApp;
```

```tsx
// Button.styles.js

const styles = {
  button: {
    color: 'var(--color-primary-500)',
    padding: 'var(--gap-m) var(--gap-s)',
  },
};

export default styles;
```
