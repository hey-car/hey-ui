# Installation 🛠

---

In order to start using **heycar-uikit**, please add this project
as a dependency of your project with

`npm install --save @heycar-uikit/button`

### Importing components

You can import components to your project with

`import { Button, Input } from '@heycar-uikit/core';`

and use them like so

```
  const component = () => (
    <div>
      <Button variant="primary" onClick={handleOnClick}>Do something</Button>
      <Input placeholder="Write something" onClick={handleInputClick} />
    </div>
  )
```
