## the global design system for **heycar**

**heycar-uikit** is a reusable component library that helps **heycar** developers build consistent and memorable user interfaces.

Along with **heycar's** constant international growth, came the strong demand for a library that can be used throughout the markets.

With **heycar-uikit**, we are able to make our several global applications look consistent, memorable, and easy to develop.

### Goals

Below are listed the goals of **heycar-uikit**, based on the needs of our developers.

- Create a component library that supports all required browsers and that provides ready-to-use React components.
- Create a component library that contains global styles and typography, also allowing brand and theme customization.
- Create components that are simple, without any business logic, state management, or dependencies that would conflict with or dictate web applications development.
- Create components that support internationalization and are fully accessible.
- Include sufficient high-quality documentation.

### Benefits

The benefits of using **heycar-uikit** are strongly relevant for **heycar's** developers as well as for the company itself and its applications.

- Consistency across all **heycar's** international applications.
- Enhancement of the **heycar** brand with a visual identity that makes users recognize the company around the globe.
- Improvement of development time by assisting developers with consistent and easy components that can be implemented fast in any of **heycar's** web applications.
- Prevention of duplicated code.
- Delivery of quality and accessible user interfaces.
