# Theme

## Colors

Listed below are all of the color names and their following possible shades
available as CSS variables on this design system. **Each shade has
its own name**, that can be identified in between parenthesis right below
their value.

In order to use the colors of the **hey-ui** design system as
CSS variables, start the name of the variable with `--color`, followed by
the color name that you want (e.g. `-primary`, `-success`) and the shade
name (e.g. `-main`, `-hover`).

```css
background-color: var(--color-primary-surface);
color: var(--color-secondary-main);

background: var(--color-secondary-surface);
box-shadow: 10px 5px 5px var(--color-success-surface);
```
