# Theme

## Colors

Listed below are all of the color names and their following possible shades
available as CSS variables on this design system. **Some shades have
alternative names**, that can be found in between parenthesis right below
their value.

In order to use the colors of the **hey-ui** design system as
CSS variables, start the name of the variable with `--color`, followed by
the color name that you want (e.g. `-primary`, `-success`) and the shade
value or alternative name (e.g. `-500`, `-default`).

```css
background-color: var(--color-primary-300);
color: var(--color-secondary-500);

background: var(--color-secondary-default);
box-shadow: 10px 5px 5px var(--color-success-light);
```
